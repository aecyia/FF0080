# FF0080 README

+ VS CODE
+ High Contrast Black / Dark
+ Main Colours Used
  + FF0080 -- Pink
  + FFFFFF -- White
  + 000000 -- Black
  + 9900FF -- Purple
  + 00FFFF -- Blue
  + 00FF00 -- Green
  + 777777 -- Grey
  + 555555 -- Grey
  + 333333 -- Grey
  + 222222 -- Grey

## Installation

1. Download this repository
2. Move FF0080 folder to .vscode/extensions
3. Restart VS Code
4. File >> Preferences >> Theme >> Color Theme
5. FF0080
